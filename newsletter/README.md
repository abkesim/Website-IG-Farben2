# AI Newsletter

A daily AI newsletter auto-generated by Claude, covering:

- **Data center power supply** — energy deals, infrastructure, power consumption
- **Local / open-source models** — new releases for Ollama, llama.cpp, HuggingFace, etc.
- **Big company releases** — Google, Anthropic, OpenAI, Meta, xAI, Mistral, and more

Each issue is split into three sections: *What's New Today*, *What's New This Week*, and *What's New This Month*.

---

## Setup

### 1. Add your Anthropic API key

Go to your repository → **Settings → Secrets and variables → Actions → New repository secret**

| Name | Value |
|------|-------|
| `ANTHROPIC_API_KEY` | your key from console.anthropic.com |

### 2. Enable Actions

Make sure GitHub Actions is enabled in your repository settings.

### 3. Run manually (optional first test)

Go to **Actions → Daily AI Newsletter → Run workflow** to trigger it immediately.

After that it runs automatically every day at **7:00 AM UTC**.

---

## Output

Each newsletter is saved as `newsletters/YYYY-MM-DD.md`.  
An index of all issues is kept at `newsletters/README.md`.
